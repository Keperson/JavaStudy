Операции с целыми числами

Все операции делятся на два типа:

  • унарные
  • бинарные.

К унарным относятся операции, в которых участвует один операнд. В бинарных операциях участвуют два операнда.
Операнд – это данные, которые принимают участие в операции.

 

Бинарные операторы


Например, оператор сложения + – бинарный: \( 2+3 \) – операндами являются числа \( 2 \) и \( 3 \).



Список бинарных арифметических операций:

  1. сложение: \( a + b \)
  2. вычитание: \( a - b \)
  3. умножение: \( a * b \)
  4. деление
    1. нацело: \( a / b \)
    2. остаток от деления: \( a \) \( \% \) \( b \)


Операции +, -, * такие же, как в математике. Деление в программировании делится на два разных деления: нацело и с остатком. При делении двух целых чисел результатом также будет целое число. Например, при делении \( 19 / 5 \) результатом будет число \(3\).

Оператор \( \% \) возвращает остаток от деления. Результатом операции \(19 \) \( \% \) \( 5 \) будет \( 4 \). Стоит заметить, что \( 5\%9=5 \).

Для лучшего понимания операций рассмотрим пример:

static void Main(string[] args)
{
    int a = 5;
    int b = 2;
    Console.WriteLine("a =  " + a + ", b = " + b);
 
    int result = a + b;
    Console.WriteLine("Сложение, a + b =  " + result);
 
    result = a * b;
    Console.WriteLine("Умножение, a * b =  " + result);
 
    result = a / b;
    Console.WriteLine("Деление, a / b =  " + result + " a и  b - целые числа, деление только нацело");
 
    int resultMod = a % b;
    Console.WriteLine("Деление с остатком, a % b =  " + resultMod + "Остаток от деления");
}

 

При вычислении выражений, содержащих несколько операций, приоритет операций такой же, как в математике. Чтобы повысить приоритет операции, используются скобки:

2 + 2 * 2 = 6 
(2 + 2) * 2 = 8

 

Унарные операторы

Унарных арифметических операторов есть всего два:

  • инкрементация «++»
  • декрементация «--» (два знака "минус")

Инкрементация увеличивает операнд на единицу, а декрементация – уменьшает на единицу.

static void Main(string[] args)
{   
    int a = 0, b = 5;   
    a++; // a = 1   
    b--; // b = 4
}
