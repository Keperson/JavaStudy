Анализ цифр числа

При помощи операции деления (/) и деления с остатком (%) можно анализировать число:

int a = Convert.ToInt32(Console.ReadLine()); // 1024

Console.WriteLine("a/10 = " + a/10); // 102

Console.WriteLine("a%10 = " + a % 10); // 4


При помощи деления с остатком (%) на \( 10, 100, 1000 \), и так далее можно получить столько последних цифр, сколько нулей в делителе.

Например:

  • при делении с остатком числа на \( 10 \), мы получим его одну последнюю цифру, так как в делителе один ноль;   
  • при делении с остатком числа на \( 100 \), мы получим его две последние цифры, так как в делителе два нуля

Для наглядности напишем программу:

int a = 1024;
Console.WriteLine("a % 10 = " + a % 10); // 4

Console.WriteLine("a % 100 = " + a % 100); // 24

Console.WriteLine("a % 1000 = " + a % 1000); // 24

Console.WriteLine("a % 10000 = " + a % 10000); // 1024


При помощи деления (/) \( 10, 100, 1000, \) 10000 действует такое же правило, только берется все, кроме последних цифр.

  • при делении  числа на \(10\), мы получим все число, кроме его последней цифры, так как в делителе \(1\) ноль;
  • при делении числа на \(100\), мы получим число без его двух последних цифр, так как в делителе два нуля.

Для наглядности снова напишем программу:

int a = 1024;

Console.WriteLine("a / 10 = " + a/10); // 102

Console.WriteLine("a / 100 = " + a / 100); // 10

Console.WriteLine("a / 1000 = " + a / 1000); // 1

Console.WriteLine("a / 10000 = " + a / 10000); // 0
