Квадрат числа

По данному целому числу, найдите его квадрат.

Формат входных данных
На вход дается одно целое число, по модулю не превосходящее 100.

Формат выходных данных
Программа должна вывести квадрат данного числа.

Примечание: с первой задачей помогу. Кратко что нужно сделать:

  1. Ввести целое число;
  2. Вычислить квадрат этого числа и записать в переменную;
  3. Вывести значение данной переменной на экран.
Все эти действия были в лекциях. Желаю удачи.
Sample Input:
5

Sample Output:
25



Memory limit: 256 MB
Time limit: 8 seconds
